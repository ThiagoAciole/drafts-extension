# Drafts - GNOME Shell Extension
**Drafts** é uma extensão minimalista para o GNOME Shell que adiciona um bloco de notas rápido e acessível diretamente na sua barra superior.

O objetivo é fornecer um espaço simples para armazenar rascunhos temporários, snippets de código ou anotações rápidas sem precisar abrir um editor de texto completo.
Esta extensão interage com a área de transferência (clipboard) do sistema. Ela fornece botões e atalhos de teclado (dentro da interface da extensão) para permitir que o usuário copie textos do bloco de notas ou cole conteúdos externos nele.
